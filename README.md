# JSON予定表
## 制作アプリの説明
- 追加した予定をカレンダー形式で表示するアプリ

## 工夫した点・こだわった点
- OutlookやGoogleカレンダーを意識したUI
- ロード時に日付を読み込み、当月のカレンダーを作成
- 前月・次月ボタンを作成し、カレンダーを更新
- 今日以降の予定は右側に一覧表示
- 予定の追加・表示を一枚のページで完結
- 予定追加後に日付・開始時間順にソートして保存

## 次回トライしたいこと
- カレンダー表示予定を３件に絞る
- 予定選択時にポップアップし、修正削除に対応させる
- デザインをもう少し意識する

## 備考
- PHPよりJSでカレンダーを自作する部分に時間がかかった
- JSON形式の取得・保存に手間取った
- DB連携までしたらユーザー認証からユーザーごとの予定表にしたい
- 未来の自分、他の人のためにコメントをたくさん書いた
